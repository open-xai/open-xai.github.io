OpenXAI

What is OpenXAI?

We introduce OpenXAI, a general-purpose lightweight library that provides a comprehensive list of functions to systematically evaluate the reliability of post hoc explanation methods. The library provides a pipeline where new datasets (both synthetic and real-world), explanation methods, and evaluation metrics can be easily integrated.
OpenXAI is an open-source library for evaluating and benchmarking post hoc explanations

Easy to Code

OpenXAI library is minimally dependent on external packages and can benchmark explanation methods with just 10 lines of code.

Easy to Evaluate

OpenXAI integrates a wide range of evaluation metrics, including faithfulness, stability, and fairness metrics.

Easy to Benchmark

OpenXAI provides an intuitive abstract template with dataloaders, trained models, and XAI-ready datasets to reliably benchmark explaination methods.
